# 🏛️ Pensionados MX v13

Sistema de gestión de beneficiarios de programas sociales.

## 🚀 Usar la app

👉 **[Abrir app](https://TU_USUARIO.github.io/pensionados-mx/)**

## 📱 Instalar como app (PWA)

### iPhone / iPad
1. Abre la URL en **Safari**
2. Toca el ícono de compartir 📤
3. Selecciona **"Agregar a pantalla de inicio"**
4. Toca **"Agregar"**

### Android
1. Abre la URL en **Chrome**
2. Toca el menú ⋮ → **"Instalar app"**
   _(o espera el banner que aparece automáticamente)_

## ✨ Funciones v13

- ✅ Gestión de beneficiarios (PAM, PCD, JCF, MT, BBJ)
- 🔍 Búsqueda con eliminación individual y masiva
- 📂 Importación CSV inteligente con detección de cambios
- 🎨 6 temas de color + color de acento personalizable
- 🔡 Tamaño de fuente ajustable
- 📊 Estadísticas y gráficas
- 🚗 Agenda de visitas
- 📅 Calendario
- 💾 Exportar Excel / CSV
- 🔒 Respaldo y restauración JSON
- 📡 **Funciona sin internet (PWA)**
